Q-2. What is a Bootstrap Container, and how does it work?

Bootstrap The most fundamental layout component in Bootstrap is a container. Bootstrap Containers are fundamental building blocks of bootstrap that enclose the content of a page. It is in charge of positioning and aligning material inside of it in accordance with the viewport or specific device. The container class (.container) contains definitions for containers. In other words, we may say that the width of the containers determines how much content is presented in the layout. The container is filled with additional components and material.

There are many uses for containers, including:

  • It needs to be used with the standard grid system.
  • to determine the layout's width and provide web content.
  • for any web project to have responsive fixed behaviour.
  • To configure the content margin for your layout's responsiveness.

The ".container" and ".container-fluid" classes from Bootstrap are the default or predefined classes for layout. The purpose of containers is to protect, centre, and occasionally pad the content they hold. Although containers can be nested, most layouts don't call for them.

Basically, bootstrap offers three different kinds of container classes:

    1. Default-Container(container)
    2. Along with sm, md, lg, xl, and xxl, responsive-container is available.
    3. Fluid-Container(container-fluid)

1. Default-Container: The '.container' class is used by the default container. It offers a fixed-width container that is responsive.

Syntax:

<div class="container">
<!-- Content here -->
</div>

2. Responsive-Container: Responsive containers are by definition responsive. You can specify a class in responsive containers that will be wide up until the specified breakpoint is reached, at which point we will apply max-widths for all subsequent breakpoints.

Syntax:

<div class="container-sm">Wide for small breakpoint</div>
<div class="container-md">Wide for medium breakpoint</div>
<div class="container-lg">Wide for large breakpoint</div>
<div class="container-xl">Wide for extra large breakpoint</div>
<div class="container-xxl">Wide for extra large breakpoint</div>

3. Fluid-Container: The '.container-fluid' class is used by fluid-container. It is utilised for a container that fills the viewport's entire width.

Syntax:

<div class="fluid-container">
<!-- Content here -->
</div>

grid structure We learned about Bootstrap and its installation in the previous article.But for the purposes of this tutorial, let's start by studying Bootstrap. In this tutorial, we'll discuss the Bootstrap Grid System.

Bootstrap's grid system permits up to 12 columns to run the width of the page. You can use each one separately or combine them to create wider columns. Grid Classes: grid system features four classes that can be combined to create layouts with additional flexibility:

  • lg (>=992px): For Small-sized Desktops/Laptops
  • xl (>=1200px): For Larger-sized Desktops/Laptops

Note: Depending on the size of your screen, the output may vary slightly from what is seen.

A thumbnail is a little image that serves as a representation of a larger image. Thumbnails in Bootstrap make this simple to accomplish. To display connected images in grids (grid system), Bootstrap's.thumbnail class is utilised. A thumbnail is produced using class.thumbnail within the element a>. The classes.col-sm-* and.col-md-*, where * denotes a number, are used to arrange images into grids.

Syntax:

<div class="d-inline-flex p-2"></div>
